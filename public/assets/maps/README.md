# West Bengal Election Map

## Required File

This folder should contain:
- `wb-election-map-2021.png` - Official 2021 West Bengal Legislative Assembly Election Map

## Download Instructions

### Option 1: Wikimedia Commons (Recommended)
1. Visit: https://commons.wikimedia.org/wiki/File:2021_West_Bengal_Legislative_Assembly_election_map_by_parties.svg
2. Click the download button or right-click the image and "Save image as..."
3. Convert to PNG format if needed (save as PNG)
4. Save the file as `wb-election-map-2021.png` in THIS directory
5. Refresh the West Bengal Map Dashboard page

### Option 2: Election Commission of India (Official)
1. Visit: https://eci.gov.in/statistical-report/statistical-reports
2. Look for West Bengal 2021 Legislative Assembly election reports
3. Download the official election map
4. Save it as `wb-election-map-2021.png` in THIS directory
5. Refresh the West Bengal Map Dashboard page

## File Location

Place the downloaded file here:
```
public/assets/maps/wb-election-map-2021.png
```

## Map Specifications

- **Format**: PNG (recommended for best quality)
- **Name**: Must be exactly `wb-election-map-2021.png`
- **Content**: 2021 election results showing all 294 West Bengal assembly constituencies
- **Dimensions**: At least 1200px width recommended for clarity
- **License**: CC BY-SA 4.0 (from Wikimedia Commons)
